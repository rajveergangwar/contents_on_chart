Contents on chart

Drupal 7 module to show various reports on chart

View on GitHub Download .zip Download .tar.gz

Drupal 7 module to show report of nodes & content types on chart pie chart , line chart , bar chart , column chart ,area chart

by @rajveergangwar

This module display animated charts of various data

  1. date wise nodes on chart
  2. database logs on chart
  3. node revision on chart
  4. user wise nodes on chart

Contents on chart drupal 7 module includes following charts

  1. pie chart
  2. line chart
  3. bar chart
  4. column chart
  5. area chart

This drupal 7 module you can use for reports on chart

Basic steps to check administrator:

  • install module
  • enable module
  • goto to admin/config

you will see following menus

: - Form for Nodes Display (this is the form which you have to user to generate the chart)

: - Database Logs (database logs on chart)

: - User wise nodes (user's nodes published chart)

: - Display date wise nodes (date wise nodes chart)

: - Display chart total number of nodes in diffrent content types (content types's nodes on chart)

: - revision for nodes (revision of nodes on chart)
